# Jenkins VM

This is a Jenkins VM based on Vagrant.

Jenkins host configuration:
* IP: 192.168.33.80
* Port: 8080

Start VM with `vagrant up`

Once running, you can access Jenkins opening [http://192.168.33.80:8080/](http://192.168.33.80:8080/)
